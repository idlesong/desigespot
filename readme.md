# designspot
设计师的辅助网站

### Features
1. 设计师作品中心（上传，展示，交流）
2. 设计师的素材库，类似一个商城，可以提供类似3D等资源，和线下的导航
3. 设计交流论坛

### brain storm
1. 商城产品为模型的设计交流指导，这个可以作为主要卖点。
1. 如何选，如何配出各种风格是最主要的需求；算出成本。
1. 效果图直接链接到商城产品如何？直接算出预算？
1. 软装素材中的经典案例，直接链接到商城
